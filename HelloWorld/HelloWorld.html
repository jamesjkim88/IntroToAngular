What is ... ?

  • angular.module

    angular.module is a colleciton of angular services, directives, controllers, filters, and configuration information. angular.module takes two arguments, the module name (in which case we used "app") and an array of dependencies. (for this application we do not require any depdencies, but still require to put in an empty array; if not you will get Uncaught Errors.)

  • controller/ng-controller:

  • controller: controller is the logic or the brain of the application
  • ng-controller: an angular directive that attaches the controller from the js file to your html
    you can add your ng-controller inside appropriate html tags.

  • $scope:

    Is an object that refers to the model application. $scope is where you can add values and functions and attached to
    to the DOM to bind to the View.
