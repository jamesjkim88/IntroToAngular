What is Two-Way Binding in AngularJS?

  • Two Way Binding always syncs the model and view at the same time. Meaning, if there is a change in the model, the view would change according to the model change and visa versa.

  • In order for two way binding to work, we need to use ng-model into our html page. Here is an example:
    <input ng-model="someData">
    ng-model or ngModel (the camel case means there is a dash before the "M", just like ng-model) directive will update the model whenever the view changes.
