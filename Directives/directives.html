ngHide and ngShow

ngShow is working

You see how the button hides and the "ngShow is working" shows up and on top of that we really didnt need to write anything on the JavaScript side.

ng-model will bind whatever text is in the input bar and ng- show and ng-hide directives will see if there is a text in the input

So pretty much ng- show and ng-hide checks if "string" is being binded if it is then show if not then hide
